Kinect的标定

标定相机的内参

下面是这个单目相机的标定方法，注意相机标定是一个非常需要耐心的工作，切忌在相机拍摄的过程中发生相对移动，会导致识别算法识别的棋盘格不准，导致严重的标定错误。

https://blog.csdn.net/qq_30460905/article/details/8039057ROSr6

我标定的是Kinect2，Kinect2其实有一套自己的标定包，在iai_kinect2(这个包就是建立Kinect2的驱动和ROS的bridge)

Kinect的标定包括，标RGB图，标IR（红外相机）图，红外相机RGB相机对齐的操作。

https://github.com/code-iai/iai_kinect2/tree/master/kinect2_calibration

**realsense相机 **实验室有的是D435i和SR300这两款相机在相机内部都有已经标好的内参，因此不需要重新标定

预先准备

libfreenect2安装好，./build/bin/Proto..测试成功

iai_kinect2编译。

连接相机。

打印棋盘格，棋盘格连接：

https://github.com/code-iai/iai_kinect2/blob/master/kinect2_calibration/patterns/chess5x7x0.03.pdf

标定顺序

代码全是rosrun，需要先roscore把节点先启动起来

  1. If you haven’t already, start the kinect2_bridge with a low number of frames per second (to make it easy on your CPU): 
    rosrun kinect2_bridge kinect2_bridge _fps_limit:=2
  2. reate a directory for your calibration data files, for example: 
    mkdir ~/kinect_cal_data; cd ~/kinect_cal_data
  3. ecord images for the color camera: 
    rosrun kinect2_calibration kinect2_calibration chess5x7x0.03 record color
  4. alibrate the intrinsics: 
    rosrun kinect2_calibration kinect2_calibration chess5x7x0.03 calibrate color
  5. Record images for the ir camera: 
    rosrun kinect2_calibration kinect2_calibration chess5x7x0.03 record ir
  6. Calibrate the intrinsics of the ir camera: 
    rosrun kinect2_calibration kinect2_calibration chess5x7x0.03 calibrate ir
  7. Record images on both cameras synchronized: 
    rosrun kinect2_calibration kinect2_calibration chess5x7x0.03 record sync
  8. Calibrate the extrinsics: 
    rosrun kinect2_calibration kinect2_calibration chess5x7x0.03 calibrate sync
  9. Calibrate the depth measurements:
    rosrun kinect2_calibration kinect2_calibration chess5x7x0.03 calibrate depth
  10. Find out the serial number of your kinect2 by looking at the first lines printed out by the
    kinect2_bridge. The line looks like this: device serial: 012526541941
  11. Create the calibration results directory in kinect2_bridge/data/$serial:
    roscd kinect2_bridge/data; mkdir 012526541941
  12. Copy the following files from your calibration directory (~/kinect_cal_data) into the directory you just created: calib_color.yaml calib_depth.yaml calib_ir.yaml calib_pose.yaml
  13. Restart the kinect2_bridge and be amazed at the better data.

按键

Windows:

ESC, q: Quit

SPACE, s: Save the current image for calibration

l: decrease min and max value for IR value rage

h: increase min and max value for IR value rage

1: decrease min value for IR value rage

2: increase min value for IR value rage

3: decrease max value for IR value rage

4: increase max value for IR value rage

Terminal:

CRTL+c: Quit

标定流程

运行上面的标定顺序中的命令行，格式如下，其中xxxxx代表的是color depth sync这三个标定过程。

rosrun kinect2_calibration kinect2_calibration chess5x7x0.03 record xxxx

会弹出监视窗口，标定板放置要近，使用三脚架固定，每个交叉点检测清楚无跳动，然后按空格保存，拍摄100张图像，

运行上面的标定顺序中的命令行，格式如下。

rosrun kinect2_calibration kinect2_calibration chess5x7x0.03 calibrate xxxxx

计算标定结果，

以上过程重复3次分别标定红外相机，RGB相机，两个相机对齐的方法。

注意：

一定要观察图像上的小点，必须保证每个小点都点在棋盘格交点处，两排之间有斜线连接，然后才能按空格保存

相机和标定板有距离限制，太远就识别不出来了。

每一个数据的record，拍摄100张以上的图像，

计算过程时间比较长。

拍的时候一定要保证标定板不要动，最好用三脚架固定。
